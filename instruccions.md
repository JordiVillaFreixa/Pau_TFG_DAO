# TFG_DAO

## Instalacio amber

```
tar xvf Amber22.tar 
tar xvf AmberTools23.tar 
mv amber22_src/ ../software/
cd ../software/amber22_src/build
./run_cmake
make install

```
## Quan està instal·lat l'amber, cada cop que vull treballar amb ell he de còrrer la següent comanda:

```
cd $AMBERHOME
source amber.sh
cd

```
